Font System

The foundation of interface design

The font system is one of the most foundational parts of any interface design. Text is a major channel for users to understand application content and complete their work, and a well designed font system will greatly enhance the user's reading experience and work efficiency.

The Ant Design typography system is based on the design principle of "dynamic order" combined with the law of natural logarithm and temperament. We strongly recommend it since it has been verified by a large number of Ant products.

Key Aspects of Our Font System:

Font Family

System default fonts with fallback options for optimal readability across platforms

Base Font Size

Optimized 14px base size for optimal reading at 50cm distance

Font Scale & Line Height

10 carefully calibrated sizes with corresponding line heights

Font Family Implementation

  'Helvetica Neue', Arial, 'Noto Sans', sans-serif,
  'Apple Color Emoji', 'Segoe UI Emoji', 'Segoe UI Symbol',
  'Noto Color Emoji';

Optimal reading distance and angle visualization

Our font scale system inspired by pentatonic scale

Font Weight

Our font weight system follows principles of order, stability, and restraint. We primarily use:

  • Regular (400) for most text
  • Medium (500) for emphasis
  • Semibold (600) for bold English text

Font Color

We maintain WCAG AAA level contrast ratios (7:1 or higher) between text and background colors for optimal readability and accessibility.

Advanced Tips

  1. Systematic Design Thinking: Plan font hierarchy consistently across the system
  2. Less is More: Use minimal font variations to achieve visual goals
  3. Rhythmic Scale: Create visual harmony through strategic font size choices
